任务要求：

在控制台中提示输入石头、剪刀、布，按回车键，然后给出游戏结果。

分析：

我们知道在游戏规则中，石头克剪刀，剪刀克布，布克石头。但是这在计算机中并不是很好直接的表示，因此我们分别用0、1、2分别代表游戏中的石头剪刀布。

那么电脑该如何出拳呢？那就该用到python中的一个模块random中的一个方法**random.randint()**在0~2范围内产生一个随机整数，就表电脑出拳了。
